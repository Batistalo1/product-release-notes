# Version 23.07
**July 2023**

---

##  New Features

- Added OSS event route and WebSocket.
- Allows coupling devices directly from the DOM Service application.
- Allows filtering the devices to synchronize with DOMBridge in the DOM Service application.
- Changes on Day Schedules will now only impact the devices referring to that Day Schedule, not others.
- Depending on the signed-in user, the application is no longer allowed to read events from devices.

---

##  Fixes

- Credentials in OSS Updater are no longer case-sensitive.
- More special characters are now allowed in passwords for backup/restore.

---

##  Known Issues

Battery replacement is not always correctly detected and stored.
