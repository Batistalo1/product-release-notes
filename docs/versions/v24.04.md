##  New Features

- New Emergency tag added to special cards.
- New Master Key tag added to special cards.
- Creating an OSS credential with Intervention mode.
- OSS Creator supports EID (FW 5.9).
- Access Mode is not available on offline devices.
- The properties `network-subnetmask` and `network-gateway` are now included in `getDevices` with the `with-details` query parameter.

---

##  Fixes

- The "Battery Replace" OSS event will now be repeated every day for 1 week.
- Unordered DoorgroupIds no longer trigger synchronization.
- Changing a TPS name no longer triggers DOMBridge synchronization.
- A bad update no longer crashes the NetManager.
- Improvement of synchronization with the DOMService app (outbox-being-filled is now managed).
- Restore on VM correctly restores the dombox ID.
- DoorGroup can be deleted even if the OSS configuration is set on the device.
- Manufacturer data is better managed during OSS import.
- OSS name is now removed from the device when the OSS configuration is removed.
- Special cards are now loaded correctly at startup.
