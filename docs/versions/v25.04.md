# Version 25.04
**December 2025 · API v32** <span class="badge badge-latest">LATEST</span>

---

##  New Features

- Improved DOMController performances when handling a large number of online devices.
- Credential size can now be specified to the system. The DOMController will handle padding and trimming for the permissions automatically.
- New API endpoint allowing configuring several OSS doors at once.
- New API endpoint allowing configuring several device configurations at once.
- Last updated time of an OSS transponder is now available in the credential data.
- Log files' size can be customized ([1–365] days, [1–1024] MB).

---

##  Fixes

- Outdated summer periods are kept API-wise but are no longer synced to the devices.
- RFNM and OSS Updater can now be correctly discovered via the DOM Service application.
- Handles a known issue from MariaDB (versions 10.5.25, 10.6.18, 10.11.8, 11.0.6, 11.1.5, 11.2.4 and 11.4.2) on backup files.
- Improves the update process of whitelisted transponders on OSS Updater.
