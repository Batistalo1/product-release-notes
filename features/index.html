    Features

    Comprehensive catalogue of all DOMController features with detailed documentation, usage guides, and version history.

    Access & Credentials

    Configurable Credential Size

    Specify credential size with automatic padding and trimming handling

    Available in: 25.04

    Hybrid Card Support

    Select which technology must be used for hybrid cards

    Available in: 25.01

    OSS Creator

    Create OSS application data structure on virgin Desfire transponders

    Available in: 23.10

    OSS Dynamic Validity

    Set OSS dynamic validity in weeks, months or years

    Available in: 25.02

    OSS Custom Key Numbers

    Assign different key numbers when configuring OSS (out of standard)

    Available in: 25.02

    Special Cards (Emergency & Master Key)

    Emergency and Master Key tags on special cards

    Available in: 24.04

    Doors & Devices

    Door Abstraction

    Create a door without a coupled device and preconfigure it

    Available in: 25.03

    Auto-Coupling

    Automatic coupling of devices upon discovery

    Available in: 24.10

    Factory Reset

    Reset a controller to factory settings

    Available in: 24.07

    Per-Device Schedules

    Week/day schedules configurable per device instead of system-wide

    Available in: 25.02

    DOM Online Mesh Support

    Official support for DOM Online Mesh products

    Available in: 25.02

    API & Connectivity

    Wireless+

    Report transponders to Access Control Server for access decisions

    Available in: 23.10

    WebSocket Connection Limit

    Limited to 10 simultaneous WebSocket connections

    Available in: 25.02

    Certificate Chain Support

    Support for certificates with full authority certification chain

    Available in: 25.03

    Synchronization & Performance

    Fast Online Synchronization

    Reduce sync time from 15 min to 10 sec

    Available in: 23.03

    Schedule Isolation

    Changes only impact devices referring to the modified schedule

    Available in: 23.03, 23.07

    Backup & System

    PHP 8 Migration

    Migration from PHP 7 to PHP 8

    Available in: 24.10

    Device Recovery System

    Recovery system to retrieve previously coupled lost devices

    Available in: 25.01

    OSS Updater

    New device for updating OSS transponders, managed by DOMBridge

    Available in: 22.12
